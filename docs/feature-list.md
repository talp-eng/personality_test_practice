# 機能一覧 - 性格診断アプリ

## 1. 概要

ユーザーがいくつかの質問に答えることで、5種類に分類された性格タイプのうち、自分がどれに該当するかを知ることができるWebアプリケーション。

## 2. 全体機能

- **レスポンシブデザイン**
  - スマートフォン、タブレット、PCなど、あらゆるデバイスで快適に操作・閲覧できるUIを提供する。
- **クライアントサイド完結**
  - サーバーとの通信は行わず、すべてのロジックはブラウザ内で完結する。状態はページをリロードするとリセットされる。

## 3. 画面別機能

### 3.1. スタート画面

- **目的**: ユーザーにアプリの概要を伝え、診断を開始するきっかけを提供する。
- **機能**:
  - **タイトル表示**: アプリケーションのキャッチーなタイトルを表示する。
  - **概要説明**: どのような診断ができるのか、簡単な説明文を表示する。
  - **診断開始ボタン**: タップすると診断画面へ遷移するボタンを配置する。

### 3.2. 診断画面

- **目的**: ユーザーに質問を提示し、回答を収集する。
- **機能**:
  - **質問文表示**: 現在の質問文を表示する。
  - **回答選択肢**: 2〜4つの回答選択肢をボタンとして表示する。ユーザーは1回のタップで回答を選択できる。
  - **プログレス表示**: 全ての質問のうち、現在何問目かを示すインジケーター（プログレスバーなど）を表示し、ユーザーに進捗を伝える。
  - **画面遷移**: 回答を選択すると、自動的に次の質問へ遷移する。最後の質問に回答すると、結果表示画面へ遷移する。

### 3.3. 結果表示画面

- **目的**: 診断ロジックに基づいて算出された結果をユーザーに分かりやすく伝える。
- **機能**:
  - **性格タイプ名表示**: 診断結果の性格タイプの名称（例：「情熱の冒険家型」）を大きく表示する。
  - **性格タイプ詳細説明**:
    - 性格の概要や特徴を説明する文章を表示する。
    - その性格タイプと相性の良いタイプなどを表示する。（任意）
  - **もう一度診断するボタン**: タップするとスタート画面に戻り、再度診断を行えるようにする。
  - **結果をシェアするボタン**: （任意）診断結果のURLやテキストをSNS（X, Facebookなど）やクリップボードに共有する機能を提供する。

## 4. 診断ロジック

- **分類**: ユーザーの回答の組み合わせに基づき、予め定義された5種類の性格タイプのうちいずれか一つに分類するロジックを実装する。
